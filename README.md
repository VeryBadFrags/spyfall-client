## Run locally

You'll need to run the server locally. See <https://github.com/VeryBadFrags/spyfall-server#run-locally>.

### With [pnpm](https://pnpm.io)

```sh
make dev
```

Run `make` to view all available commands.

### With [npm](https://www.npmjs.com)

```sh
npm run dev
```
